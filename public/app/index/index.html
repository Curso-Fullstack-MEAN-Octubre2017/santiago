BIENVENIDO A PETSTORE

Mucho mas que un software de gestion para tu centro veterinario.

PetStore es una aplicación demo operativa realizada como parte del curso de programación Fullstack-Mean.

Entre las tecnologías usadas en el proyecto estan
Mongodb
Express
Angularjs V1.5.8
Nodejs
Socket.io
Rxjs

Para facilitar el proceso de aprendizaje de dichas tecnologías se utilizó Materializecss, minimizando así el uso de CSS para el diseño.
